# Sobre Nosotros

* Implementada la nueva página "Sobre Nosotros".
* Añadida descripción de la plataforma y misión del proyecto.
* Incorporado diseño responsive moderno con estilo visual consistente con el resto de la aplicación.

### Equipo de desarrollo

* Añadidas tarjetas individuales para cada integrante del grupo.
* Incorporado sistema de selección de miembro mediante click.
* Añadido subcomponente `TeamMemberDetailComponent` para mostrar información detallada de cada desarrollador.

### Mejoras visuales

* Reestructurado el grid del equipo para evitar deformaciones al añadir más integrantes.
* Añadidas alturas mínimas y alineación uniforme en las tarjetas.
* Mejorada la distribución responsive en escritorio y móvil.
* Ajustados estilos de avatar, nombres y roles para mantener consistencia visual.

### Modelado y estructura

* Creado modelo `TeamMember`.
* Separada la lógica de detalle del integrante en un subcomponente reutilizable.
* Añadidas tecnologías y descripciones individuales para cada miembro.
